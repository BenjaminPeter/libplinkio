pio_status_t bed_read_row ( struct pio_bed_file_t *  bed_file,
snp_t *  buffer 
)

Reads a single row from the given bed_file. Each element in the buffer will contain a SNP. The SNP will be encoded as follows: 0 - Homozygous major 1 - Hetrozygous 2 - Homozygous minor 3 - Missing value

Parameters:
bed_fileBed file.
bufferThe buffer to read into. It is asssumed that it is big enough, e.g.: SNPS_IN_ROWS: Number of individuals SNPS_IN_COLUMNS: Number of loci.
Returns:
PIO_OK if a row could be read, PIO_END if there are no more rows, PIO_ERROR otherwise.
