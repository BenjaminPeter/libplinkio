pio_status_t pio_next_row ( struct pio_file_t *  plink_file,
snp_t *  buffer 
)

Reads the next row from the bed file. Depending on the storage format, this will return either a single SNP for all individuals, or all SNPs for a single individual.

The SNPs will be encoded as follows: 0 - Homozygous major 1 - Hetrozygous 2 - Homozygous minor 3 - Missing value

Parameters:
plink_filePlink file.
bufferThe row will be stored here. Must be able to hold at least pio_row_size bytes.
Returns:
PIO_OK if the row could be read, PIO_END if we are at the end of file, PIO_ERROR otherwise.
